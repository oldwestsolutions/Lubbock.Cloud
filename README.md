# Lubbock Cloud

Modern TypeScript Next.js site inspired by Texas Tech red and Microsoft gray.

## Tech
- **Next.js** (App Router, TypeScript)
- **Tailwind CSS**

## Development
1. Install deps:
```bash
cd lubbock.cloud
npm install
```
2. Run dev server:
```bash
npm run dev
```
3. Open `http://localhost:3000`.

## Build & Start
```bash
npm run build
npm start
```

## Theming
- **Primary**: `#C8102E` (Texas Tech Red)
- **Neutrals**: Microsoft gray scale in `tailwind.config.ts` under `tech.gray`.
